Recent Work

Single Cycle and Pipelined MIPS Processor

In my Computer Architecture Course during the fall of 2022, I designed both a single cycle and five stage pipelined mips processor. It is a 32 bit system that uses 128 address instruction and data memory. They both use a register file containing 32 registers of 32 bit width.

Augmented Reality Sampling Mission Tool

In the summer of 2022, I worked at DCS Corporation. During my time here, I got the chance to design an augmented reality app that helps track information about hazards. I wrote this using Java and Kotlin and utilized Googles Geospatial API. My team at the company has now picked up where I left off on the app. They are working on implementing additional functionality that I recommended.

Computer Architecture Security Research

In early 2022, I was given the opportunity to begin research with a professor. We decided to work on implementing lightweight encryption methods to add to a pipelined processor. The purpose of this is to prevent malicious attackers from reading or changing data in memory and in the pipeline registers. To accomplish this, we have used a hardware description language called chisel. Using chisel, we can work in an object oriented environment while generating verilog files to test in Synopsys.

VHDL Component Generators

In my digitial electronics course, I had the opportunity to create and design a handful of VHDL components including adders, multipliers, matrix multipliers, read only memory, hex LEDs, and much more. I performed register-transfer level simulations on these components by using modelsim. The left image above shows a generic adder which can generate an adder that can add two binary numbers of any width. The right is a generic matrix multipler which can multiply an NxN matrix with an Nx1 matrix.

Oracle SQL Hotel Database

In my sophomore year, I took principles of database systems. I was introduced to relational and nonrelational databases. For my final project, I designed a database that a hotel company could use in Oracle SQL. I created an entity relation diagram which displays all the entities, their attributes, and how they link to each other. Then, I populated the database with data and ran queries.

Matlab Image Prediction

In the spring of 2022, I worked in a group to design a pseudo-machine learning algorithm in Matlab. Our task was to make a program that could predict what category a satelite image belonged to. The five categories were baseball diamond, beach, chaparral, forest, and river. Our algorithm seperated images into smaller tiles in which color values were averaged. Then, these average values were used to create rules for each category. When posed with a new image, the algortihm compared the new image to each category using distance values.
